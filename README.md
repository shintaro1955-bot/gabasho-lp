# ガバショ！ 統合サイト

「本物の伴走で、組織を強くする」— アナウンサー・モデル、PROアスリートが企業に伴走する「ガバショ！」のWebサイト。
**ハブ（トップ）＋2プラン詳細＋共通フォーム**の静的サイト。依存・ビルド不要。

## ページ

| ファイル | 内容 |
|---|---|
| `index.html` | 統合トップ（ハブ）。2プランへの分岐・共通価値・導入フロー |
| `announcer.html` | アナウンサー・モデル版（ピンク／月30万〜・50万〜） |
| `athlete.html` | PRO Athlete版（ネイビー×ゴールド／月300万・特別講演会） |
| `contact.html` | 共通の資料請求・問い合わせフォーム（バリデーション付き・送信先未接続） |

相互ナビ：`index ↔ announcer ↔ athlete`。全CTAは `contact.html` に集約。
`contact.html?plan=announcer` / `?plan=athlete` でプラン初期選択可。

## ローカルで開く

ファイルを直接ブラウザで開くだけでOK（ビルド不要）。簡易サーバで見るなら：

```bash
cd gabasho-site
python3 -m http.server 5500
# → http://localhost:5500
```
